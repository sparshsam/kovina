# Development

## Prerequisites

- Node.js 20+
- npm

## Setup

```bash
npm install
```

## Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Build

```bash
npm run build
```

Produces a static export in `out/`.

## Lint

```bash
npm run lint
```
