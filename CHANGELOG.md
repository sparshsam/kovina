# Changelog

All notable changes to Kovina will be documented in this file.

## [0.1.0] — 2026-06-30

### Added
- Editorial exhibition design with abstract ornament components
- Full-screen project sections with large app icons
- 18 individual app detail pages at `/apps/[slug]`
- NotoSansSymbols2 font for decorative project titles
- Complete feature content for all 18 applications
- OpenSnap app listing with icon and feature content

### Changed
- Project order: Live apps first, then In Development
- OpenReader badge from "Desktop" to "Live"
- Index redirect added for clean URL resolution

### Infrastructure
- Next.js 16.2.9 with static export
- Cloudflare Pages deployment via wrangler
- Tailwind CSS v4 with dark/light theme support
