# Open Product Family — Spacing

## Lockup Spacing

The distance between OPEN and the product name in a lockup should be **1.5× the width of the letter "O"** in the OPEN wordmark at the same size.

```
OPEN••••Reader
     ↑
     1.5× O width
```

This spacing ensures the two elements are read as a single unit while remaining visually distinct.

## Clear Space

Minimum clear space around the OPEN wordmark or lockup:

```
┌──────────────────────────┐
│       (clear space)       │
│        ┌──────┐           │
│   ░░░░░│ OPEN │░░░░░      │  ← height of O
│        └──────┘           │
│       (clear space)       │
└──────────────────────────┘
```

No other elements may intrude into this space.

## Minimum Sizes

| Medium | OPEN wordmark | Lockup (OPEN + product) |
|---|---|---|
| Digital | 40px wide | 120px wide |
| Print | 12mm wide | 36mm wide |
| Header/nav (wordmark only) | 20px wide | — |
| GitHub repository header | — | 200px wide |
| README hero | — | 400px wide |

Below these sizes, legibility degrades. Use the header variant for very small applications.
