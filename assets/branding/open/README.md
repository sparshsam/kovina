# Open Product Family Branding

This directory contains the brand identity system for the **Open Product Family** — the shared identity for every Kovina application beginning with "Open".

## Hierarchy

```
KOVINA          ← Parent ecosystem (frozen v1.0.0)
  │
  OPEN          ← Product family (this system)
  │
  Reader        ← Individual product
  Proof
  Ledger
  ...
```

Kovina is the parent brand. Open is the product family. Each application name is the individual product.

## Status

This is **Draft v1.0.0** — the first design candidate. Not yet frozen. Manual review required before becoming official.

## Contents

| Directory | Description |
|---|---|
| `wordmark/` | OPEN wordmark SVGs (master, variants, exports) |
| `lockups/` | Product lockups: OPEN + product name |
| `templates/` | Reusable templates for README, headers, etc. |
| `specifications/` | Typography, hierarchy, spacing, usage |
